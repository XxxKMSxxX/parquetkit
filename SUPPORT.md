# Support

- **Bug reports** — [open an issue](https://github.com/XxxKMSxxX/parquetkit/issues).
  A sample file that reproduces the problem (or its schema) helps enormously.
- **Questions and ideas** —
  [GitHub Discussions](https://github.com/XxxKMSxxX/parquetkit/discussions).
- **Security vulnerabilities** — see [SECURITY.md](./SECURITY.md) for private
  disclosure. Please do not open public issues for these.
- **Anything else** (press, licensing, partnerships) —
  [contact@parquetkit.com](mailto:contact@parquetkit.com).
