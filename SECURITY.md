# Security Policy

ParquetKit is a fully client-side static site: files you open are never
uploaded, and there is no server-side attack surface. Security issues are
still possible (e.g. XSS via rendered content, dependency vulnerabilities).

## Reporting a vulnerability

Please use GitHub's private vulnerability reporting on this repository
("Security" tab → "Report a vulnerability"). You should receive a response
within a few days. Please do not open public issues for security reports.
